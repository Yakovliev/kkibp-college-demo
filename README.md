# Сайт Економіко-правового фахового коледжу

Статичний базовий сайт для Економіко-правового фахового коледжу Київського кооперативного інституту бізнесу і права. Структура основних розділів звірена з файлом `docs/Сайт структура коледж.docx`.

## Що є в комплекті

- `index.html` — головна сторінка.
- `college.html`, `admissions.html`, `students.html`, `alumni.html`, `science.html`, `library.html`, `news.html` — внутрішні сторінки.
- `css/styles.css` — усі стилі, адаптивні брейкпоінти та теми день/ніч.
- `js/main.js` — випадаючі меню, мобільна навігація, пошук, перемикач теми день/ніч, рендеринг карток новин із тегами циклових комісій і фільтром, іконки Цілей сталого розвитку, toast-повідомлення.
- `js/news-data.js` — спільне джерело даних новин (заголовок, дата, excerpt, зображення, теги циклової комісії, цілі сталого розвитку).
- `assets/sdg/` — іконки 17 Цілей сталого розвитку для сторінок новин.
- `assets/fonts/inter/` — локальні файли шрифту Inter і ліцензія OFL.
- `previews/laptop.html`, `previews/tablet.html`, `previews/mobile.html` — сторінки для перегляду у фіксованих розмірах пристроїв.
- `docs/technical-spec.md` — прийняті вимоги для майбутньої CMS або production-реалізації; WordPress у ТЗ згадується як робоча гіпотеза, а не фінально затверджена платформа.
- `docs/decisions/` — зафіксовані дизайн-рішення демоверсії.

## Запуск

Сайт можна відкрити подвійним кліком по `index.html`. Для коректної локальної розробки зручніше запустити простий сервер:

```bash
python -m http.server 8000
```

Після цього відкрити `http://localhost:8000`.

## Адаптивність

- ноутбук / desktop: від 1100 px;
- планшет: 720–1099 px;
- мобільний: до 719 px.

Одна кодова база автоматично підлаштовується під ширину екрана. На desktop використовується mega-menu, на планшеті й мобільному — висувне меню з акордеонами.

## Типографіка

Основна гарнітура сайту — **Inter**. Вона підключається локально з `assets/fonts/inter/`, а базовий стек у CSS має залишатися таким:

```css
font-family: Inter, system-ui, sans-serif;
```

Файли шрифту:

- `assets/fonts/inter/InterVariable.woff2`;
- `assets/fonts/inter/InterVariable-Italic.woff2`;
- `assets/fonts/inter/OFL.txt`.

Не підключати Google Fonts, CDN або додаткові платні/proprietary шрифти без окремого дизайн-рішення. Прийняте рішення зафіксоване в `docs/decisions/DDR-016-local-inter-typography.md`, а вимоги до майбутньої реалізації — у `docs/technical-spec.md`.

Назви закладу та основні контакти винесені в HTML-шаблони сторінок і можуть бути уточнені без зміни структури сайту.
