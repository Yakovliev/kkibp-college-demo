# Сайт Економіко-правового фахового коледжу

Статичний базовий сайт для Економіко-правового фахового коледжу Київського кооперативного інституту бізнесу і права. Структура основних розділів звірена з файлом `docs/Сайт структура коледж.docx`.

## Що є в комплекті

- `index.html` — головна сторінка.
- `college.html`, `admissions.html`, `students.html`, `alumni.html`, `science.html`, `library.html`, `news.html` — внутрішні сторінки.
- `css/styles.css` — усі стилі та адаптивні брейкпоінти.
- `js/main.js` — випадаючі меню, мобільна навігація, пошук, фільтри, toast-повідомлення.
- `previews/laptop.html`, `previews/tablet.html`, `previews/mobile.html` — сторінки для перегляду у фіксованих розмірах пристроїв.

## Запуск

Сайт можна відкрити подвійним кліком по `index.html`. Для коректної локальної розробки зручніше запустити простий сервер:

```bash
python -m http.server 8000
```

Після цього відкрити `http://localhost:8000`.

## Адаптивність

- ноутбук / desktop: від 1100 px;
- планшет: 720–1099 px;
- мобільний: до 719 px.

Одна кодова база автоматично підлаштовується під ширину екрана. На desktop використовується mega-menu, на планшеті й мобільному — висувне меню з акордеонами.

Назви закладу та основні контакти винесені в HTML-шаблони сторінок і можуть бути уточнені без зміни структури сайту.
